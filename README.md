# Knightingale

A minimal voice dictation daemon for Linux, Windows, and macOS. Bring-your-own-key cloud STT or fully local Whisper inference. Single static binary.

Work in progress — see [`docs/`](https://shatilkhan.github.io/knightingale/) for setup once published.
